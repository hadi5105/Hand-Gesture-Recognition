# ✋ Hand Gesture Recognition

A real-time Hand Gesture Recognition and Finger Counting project built with Python, OpenCV, and MediaPipe.

## 🚀 Features

- Real-time hand detection
- Finger counting
- Webcam support
- Fast and accurate tracking

## 🛠 Technologies

- Python
- OpenCV
- MediaPipe

## Installation

```bash
pip install opencv-python mediapipe
```

## Run

```bash
python hand.py
```

## Demo

This project detects a hand using a webcam and counts the number of raised fingers in real time.
